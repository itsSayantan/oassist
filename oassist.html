OAssist - Office Assistant
Developer: Sayantan Ghosh
Project link
Description
OAssist - Office Assistant is a office utility application made for M/S. S.A. Construction. This application is made to be used as a Google Chrome Extension.
Requirements
This application requires Google Chrome web browser and internet when hosted on an external server.
Installation Instructions
Goto: More Tools > Extensions. Enable Developer mode and click on Load unpacked extension... button. Then select the oassist folder from the location you chose when you downloaded OAssist. Open a new tab in the browser to start using OAssist.
